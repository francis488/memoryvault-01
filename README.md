# 🔒 MemoryVault - Private Secure Vault

MemoryVault is a lightweight, browser-based personal diary and media vault. It allows users to save text memories along with photos, videos, and audio files securely using local browser storage.

## ✨ Features
- **Local Encryption:** Uses IndexedDB for high-capacity local storage.
- **Media Support:** Upload and preview images, videos, and audio clips.
- **Category Filtering:** Organize memories by Personal, Work, or Family.
- **Search:** Quickly find memories using the built-in search bar.
- **Privacy First:** Data never leaves your browser; no server-side storage is used.
- **Backup & Restore:** Export your entire vault to a JSON file and import it on other devices.

## 🚀 How to Use
1. **Entry:** Start at the gateway and create a local account.
2. **Dashboard:** Access your private vault to add new memories.
3. **Storage:** Monitor your browser's storage limit via the storage bar.
4. **Security:** Use the logout button to clear the active session.

## 🛠️ Technology Stack
- **Frontend:** HTML5, CSS3 (Inter font-family)
- **Logic:** Vanilla JavaScript
- **Database:** IndexedDB (Client-side)
- **Auth:** LocalStorage API
